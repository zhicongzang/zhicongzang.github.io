Gwent昆特牌牌组分享

Posted on 2017-04-22 | In Game | |

昆特牌国际服入坑将近一个月了，排位一路飙升到3400+基本都是非常顺利的。这里分享下我常用的牌组和一些心得。

松鼠党控制牌组

卡组介绍

这是一套基本的控制松鼠卡组。通过松鼠党天然的技能优势赚取卡差，加上核心Combo：Toruviel -> Operator -> Morenn的双BUFF无解大哥一波流取得胜利。在面对各方势力时都没有什么短板。是一套非常均衡稳定的强力卡组。

单卡分析

Bronzes

Vrihedd Dragoon：5+4非常优秀的超模铜卡，通常最理想的是用来BUFF Toruviel，但是如果手里没有Toruviel，Vrihedd Sappers也是不错的BUFF对象。
Elven Mercenary：同样是非常优秀的铜卡，随便拉出什么法术基本上都能打出8+的超模点数，而且还能压缩卡组。需要注意的是万一拉出Scorch会不会烧掉自己的卡牌，还有就是拉出Lacerat会不会浪费了。
Hawker Support：虽然本身亏2点，但是配合法术有着很高的成长空间。
Vrihedd Sappers：备用BUFF对象。需要注意必须在倒数第二回合之前打出，否则牌没翻出来就尴尬了。
Lacerate：非常好用的追点卡牌，一般在对抗怪物，松鼠和北方的时候能打出巨额伤害。
First Light：在对抗Dagon或者群岛时可以留一张在手牌，否则在开局换牌时都换掉。这样可以配合Elven Mercenary进一步的压缩卡组。
Alzur’s Thunder：用来解对手关键卡牌。大多数的关键卡牌都能被Alzur’s Thunder解掉。

Silvers

Toruviel：核心卡牌，BUFF的首选单位，尽量留到第三回合。除了Morenn和Villentretenmerth，被BUFF过的Toruviel就是一张无解的大哥。
Roach：压缩卡组神级银卡，相当于金卡自带+3的效果。尽早的使用金卡将Roach从牌组里拉出来，以防第二三轮摸牌的时候摸到手里。配合Milva第一回合可以赚到卡差（第一回合空下Milva，拉出Roach，Roach再返回手牌，对手没牌可以返）。
Operator：这套卡组的核心卡牌之一。当Morenn可以被打出的时候（在手中或者可以用领袖技能拉出来时）复制自己被BUFF过的牌（Vrihedd Dragoon或Elven Mercenary），再用Morenn来破坏对手获得的复制牌。在对抗松鼠党时，可以通过动画效果来定位复制牌在对方手牌中的位置。防止对方用其他牌来骗Morenn。
Morenn：配合Operator打出Combo。需要注意的是怪物的Caretaker，在对抗怪物时尽量防止不要被对手的Caretaker从墓地里拉出Morenn。
Yaevinn：抽卡间谍，压缩卡组，赚卡差，防Scorch。
Scorch：在有好的时机（一烧二，一烧n的）的时候不要犹豫，有的时候犹豫了，机会就过去了。但是在对抗怪物的时候，要考虑对手Griffin，不要过早的用掉Scorch。

Golds

Aglaïs：最强金卡之一，两次Scorch不是一般的强。当然有的时候拉Lacerate清场或者Alzur’s Thunder摧毁对手的关键卡牌也是很不错的。
Milva：多功能卡牌，即可以用来赚卡差，又可以解对面BUFF过的大哥，还可以用来回收Vrihedd Dragoon继续BUFF大哥。
Isengrim：最强金卡之一，配合法术，强力铺场。
Saskia：场上有超过5个非金单位就能超模。配合Isengrim通常能打出爆炸输出。

卡组Combos

核心：Toruviel（Vrihedd Sappers） -> Operator -> Morenn
Milva -> Roach
Milva -> 对手的抽牌间谍
Hawker Support -> Isengrim -> Elven Mercenary（法术） -> Saskia

技巧心得

换牌阶段

优先更换顺序：Roach -> First Light -> Vrihedd Sappers -> Alzur’s Thunder -> Hawker Support
保证核心Toruviel和Vrihedd Sappers至少有一张在手中，可以将多余的换掉。
优先保证手中+领袖技能能打出核心combo。
在对抗Dagon时手牌中可以留一张First Light，其余的都可以换掉。
在对抗吞噬怪物或者铺场北方时，可以留Alzur’s Thunder用来解对方关键卡牌。

First Round

先下Hawker Support，Isengrim。用Elven Mercenary来压缩牌库成长Hawker Support，铺场。尽量取得第一局的胜利，亏不多于一张卡胜，这样在第二三局能够更好的发挥松鼠党的优势（后两局都是后手）。如果不能轻松获胜，尝试让对手亏超过一张牌放弃第一局。控制松鼠赢的就是卡差。

Second Round

如果赢了第一局，松鼠党的技能可以帮助你在这一场获得巨大卡差优势。对手这把必须赢下，不然游戏就结束了，但是在先手的情况下，赢下输不起的局要花费的代价可不是一般的大。这时己方用尽量小的代价消磨对方的combos和核心卡牌。同时在第二局可以下Operator复制BUFF大哥，为第三局的双大哥做下铺垫。

如果第一局没有赢，因为手握后手及卡差优势的情况下，对手也不会在这局消耗过大精力。本局目标是在取胜的前提下尽量保存实力为第三局做准备。

Third Round

由于松鼠党的技能，第三局肯定是有后手优势的。一切顺利的话，往往还能有卡差优势。这时如果BUFF大哥是Toruviel，除了松鼠党，基本对手很少有能力处理这样的大哥。如果BUFF大哥是 Vrihedd Sappers，记得在倒数第二回合及之前放下，否则来不及翻牌就麻烦了。在对抗怪物或者群岛时，留一张First Light对抗天气还是很有必要的。尽量把Aglaïs留在第三局，面对不同的局面选择拉出不同的法术获取优势。
